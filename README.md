## Instalando Portfólio

Para instalar o **Portfólio**, siga estas etapas:

Linux and macOS:

```bash
sudo git clone https://github.com/Niinh/portfolio-pessoal-cv
```

Windows:

```bash
git clone https://github.com/Niinh/portfolio-pessoal-cv
```
